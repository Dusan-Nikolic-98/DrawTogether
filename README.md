## DrawTogether

DrawTogether is a full-stack collaborative drawing platform designed to explore modern TypeScript architecture across frontend and backend environments.

The project combines a Vue 3 + Vite frontend with a Dockerized Node.js + Express + PostgreSQL backend, emphasizing clean structure, API-driven design, and scalable project organization.

## Features

- User authentication (register / login)
- Real-time collaborative drawing (via WebSockets)
- Chat option while drawing together
- Image storage & retrieval
- Gallery view
- Individual image view
- REST API architecture
- Dockerized backend & database
